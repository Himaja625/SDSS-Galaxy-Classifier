How the Model Works

This project uses supervised machine learning models trained on the Sloan Digital Sky Survey (SDSS) dataset. Photometric features such as u, g, r, i and z band magnitudes are used to identify galaxy types.

The dataset is cleaned, normalized and balanced using SMOTE to prevent bias toward dominant classes. Multiple models are evaluated and the most accurate one is deployed.

Once deployed, users can input galaxy parameters and receive real-time predictions through the trained model.
